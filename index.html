Java Collections Framework

Изучение коллекций Java на практических примерах

Обзор Java Collections Framework

List (Списки)

  • ArrayList - быстрый доступ по индексу
  • LinkedList - быстрая вставка/удаление
  • Deque - двусторонняя очередь

Set (Множества)

  • HashSet - быстрый поиск
  • LinkedHashSet - порядок вставки
  • TreeSet - сортированное множество

Map (Карты)

  • HashMap - быстрый доступ по ключу
  • LinkedHashMap - порядок вставки
  • TreeMap - сортированная карта

Queue (Очереди)

  • PriorityQueue - очередь с приоритетом
  • ConcurrentLinkedQueue - многопоточная
